# Personal Portfolio

Cinematic WebGL portfolio for Rakati Venu Kumar built with React, Vite, Three.js, and React Three Fiber.

## Experience

- Scroll-driven cinematic camera journey from a calm night surface into a futuristic digital ocean.
- Holographic hero typography with reflected treatment, underwater data particles, neural node structures, and glowing algorithm forms.
- Glassmorphism content panels for about, skills, projects, experience, resume, and contact.
- Interactive project modal cards and a terminal-style experience section with typing animation.

## Stack

- React + TypeScript + Vite
- Three.js with React Three Fiber and Drei
- Custom CSS for motion, glow, glassmorphism, and responsive layout

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
